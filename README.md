## 네트워크 접속
 - 아이디 : root
 - 패스워드 : linux


## RetroArch 단축키
 - 핫키 활성화: 선택(누르고 있기)
 - 종료: 시작(두 번 누르기)
 - 메뉴: X
 - FPS 표시/숨기기: Y
 - 상태 저장하기: R1
 - 상태 불러오기: L1
 - 되감기: L2
 - 빨리감기: R2

참고: 일부 장치에서는 X와 Y가 반전되며 이는 정상적인 동작입니다.
